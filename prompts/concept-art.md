# Concept Art & Fantasy Prompts / 概念艺术与奇幻提示词

> Concept art, fantasy, and sci-fi prompts for Midjourney V7 — character design, environment art, and worldbuilding.

---

### Prompt 1: Futuristic Transport Hub
**Parameters:** `--ar 21:9 --v 7`
**Prompt:**
```
Futuristic conceptual architecture, a transportation hub with aerodynamic carbon fiber arches and ethereal light tunnels, speculative design, bio-inspired structures, neon glow, foggy atmosphere, epic scale, concept art, Unreal Engine 5 render --ar 21:9 --v 7
```
**Best for:** Environment concept art, sci-fi worldbuilding, architectural visualization.
**Tips:** The ultra-wide 21:9 aspect ratio emphasizes the epic scale of the architecture. "Speculative design" and "bio-inspired structures" push the output beyond generic sci-fi into thoughtful futuristic design. Naming "Unreal Engine 5 render" cues V7's understanding of high-fidelity real-time rendering aesthetics. "Foggy atmosphere" adds depth and mystery.

### Prompt 2: Dark Fantasy Poster
**Parameters:** `--ar 16:9 --s 250 --v 7`
**Prompt:**
```
Create a dark fantasy poster with a young red-haired woman surrounded by mystical elements in a gloomy setting, gothic aesthetic, dramatic lighting --ar 16:9 --s 250 --v 7
```
**Best for:** Fantasy poster design, gothic character portraits, dark-themed book covers.
**Tips:** The `--s 250` stylize value balances artistic flair with coherent poster composition. "Gothic aesthetic" combined with "dramatic lighting" ensures deep shadows and moody highlights. "Poster" in the prompt tells Midjourney to design for a strong central focal point with atmospheric framing, rather than a flat scene.

### Prompt 3: 1970s Fantasy (Frazetta Style)
**Parameters:** `--ar 2:3 --s 400 --v 7`
**Prompt:**
```
1970s dark fantasy movie poster style, barbarian warrior on a throne, Frank Frazetta inspired, oil painting texture, warm amber tones, heavy shadows --ar 2:3 --s 400 --v 7
```
**Best for:** Classic fantasy illustration, poster art, retro fantasy book covers.
**Tips:** Referencing "Frank Frazetta" targets a very specific painterly style with muscular forms and dramatic composition. The high `--s 400` stylize value amplifies the oil painting texture and rich amber palette. The 2:3 portrait ratio matches classic movie poster dimensions. "Heavy shadows" ensures the chiaroscuro effect central to Frazetta's work.

### Prompt 4: Character Consistency Test
**Parameters:** `--ar 4:5 --v 7`
**Prompt:**
```
A young girl with short black hair and green eyes wearing a yellow raincoat, holding a lantern in a forest, children's book illustration style, soft magical lighting, detailed background --ar 4:5 --v 7
```
**Best for:** Character design consistency testing, children's book illustration, character reference sheets.
**Tips:** This prompt is designed for consistency testing — run it multiple times to evaluate how well V7 maintains the character's key traits (short black hair, green eyes, yellow raincoat) across generations. "Children's book illustration style" keeps the aesthetic soft and approachable. The 4:5 ratio is ideal for character-focused compositions.

![Example](../images/concept-art/prompt-4-character-consistency-test.jpg)

### Prompt 5: Cyberpunk Character Design
**Parameters:** `--ar 16:9 --style raw --v 7`
**Prompt:**
```
Cyberpunk female hacker character design sheet, multiple poses, neon pink and blue color scheme, chrome cybernetic arm, holographic UI elements, concept art, clean white background --ar 16:9 --style raw --v 7
```
**Best for:** Character design sheets, cyberpunk concept art, game character development.
**Tips:** "Character design sheet" and "multiple poses" instruct V7 to generate a turnaround-style layout rather than a single illustration. `--style raw` keeps the output clean and professional, avoiding over-stylization that would obscure the design details. "Clean white background" isolates the character for reference use. "Holographic UI elements" add genre-specific visual interest.
