## 9. --exp 表现力参数详解

> **作者:** @nickfloats (Nick St. Pierre) | **浏览:** 126,702 | **链接:** [原帖](https://x.com/nickfloats/status/1917729947474334019)

**参数说明:**
- 范围: `--exp 0` 到 `--exp 100`
- 推荐起始值: `--exp 5-25`
- 低值增加细节和动态感，高值可能导致提示词偏差

> "In small doses it really does seem to bring a bit more life into images while maintaining coherence to the prompt. In large doses it has the potential to overwhelm things a bit and decrease prompt accuracy." — @nickfloats

## 附录: 常用参数速查

| 参数 | 用途 | 常见值 |
|------|------|--------|
| `--v 7` | 版本号 | `7`, `7.0` |
| `--ar` | 宽高比 | `1:1`, `3:2`, `4:5`, `9:16`, `16:9` |
| `--sref` | 风格参考码 | 数字 ID |
| `--sv` | 风格版本 | `4`, `6` |
| `--sw` | 风格权重 | `100`, `500` |
| `--stylize` | 风格化程度 | `50`, `300`, `500` |
| `--chaos` | 随机性 | `0-100` |
| `--exp` | 表现力 | `5-25` (推荐) |
| `--raw` | 原始模式 | 无值 |
| `--p` | 个性化 | 无值 |
